## Record the video — step by step

1. Open a real Claude Code session (Terminal, not Cursor — cleaner look)
2. Start typing a real prompt you've used before. Pause. Delete. Retype. Delete. Make it feel real.
3. Type `/prompts [your query]` — let the dropdown appear naturally
4. Pick a result. Let the full text load.
5. Record with QuickTime (Cmd+Shift+5 → screen area) or loom for the cursor trail
6. Trim to 15-20 sec. No music needed — the silence makes it feel authentic.
7. Add 2-line caption on screen: "The /prompts skill. Open source. Free forever."
